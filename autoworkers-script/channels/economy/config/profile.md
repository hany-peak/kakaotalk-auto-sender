# 국제 뉴스와 경제 이슈를 비유와 숫자로 쉽게 풀어주는 채널

## 장르
경제/시사 (국제 정세, 안보, 경제 이슈)

## 톤 & 말투
- 스타일: 친한 선생님이 설명해주는 느낌
- 상세: 존댓말 베이스, 청중에게 직접 질문 ("여러분, 이게 말이 된다고 생각하십니까?"), 핵심 포인트에서 잠깐 호흡 조절

## 금지 표현
- "여러분"
- "오늘은"
- "구독 좋아요 알림설정"
- "학자들에 의하면"
- "과학적으로 증명된"
- "~했다고 합니다"
- "충격적인"

## 시그니처 표현
- "여기서 진짜 핵심이 있습니다" — 중요 포인트 강조 시
- "쉽게 비유를 들면요" — 어려운 개념이나 숫자를 일상 언어로 전환할 때

## 서사 구조
- 선호: 몰락 서사, 반전 서사 (충격 훅 → 배경 설명 → 데이터 → 비유 → 위기/반전 예고)
- 기피: 나열형 (이유: 긴장감 없이 정보만 나열되면 몰입도 떨어짐)

## 관점
항상 소비자/일반인 관점에서 이야기한다. "나 같은 사람한테 이게 무슨 영향이지?"라는 시청자의 시선으로 국제 뉴스와 경제 이슈를 해석한다.

## 정보 전달 방식
비유+데이터 혼합. 숫자와 데이터를 반드시 일상적 비유로 풀어준다.
예: "여의도~반포대교 거리쯤 되는 좁은 해협에서 세계 석유 20%가 통과합니다"
예: "밑바닥에 구멍이 뚫린 양동이에 물을 붓는 것과 다를 게 없습니다"

## 타겟 시청자
경제·국제 뉴스에 관심 있는 30~50대 직장인. 어렵고 복잡한 국제 정세를 쉽게 이해하고 싶지만 전문 용어가 가득한 기사는 부담스러운 사람들. "이 채널 보면 뉴스가 이해된다"는 반응을 목표로 한다.
